## Wordle pero Versión Dominicana

El usuario debe adivinar una palabra dominicana en un máximo de 6 intentos, usando un teclado interactivo y tiles que muestran el estado de cada letra (correcta, presente o ausente).
Las palabras tienen diferente cantidad de letras y actualmente el banco de opciones tiene 62 palabras.
